# Aditya Portfolio

A deploy-ready static personal portfolio for Aditya, focused on AI/ML, applied generative AI, agentic systems, and research-oriented projects.

## Structure

- `index.html` - single-page portfolio content
- `styles.css` - responsive dark-mode design
- `script.js` - project filtering and footer year
- `assets/ai-research-hero.png` - hero image asset
- `assets/resume.pdf` - downloadable resume placeholder

## Customize

Update these values before publishing:

- Replace `aditya@example.com` in `index.html` with your real email.
- Replace the LinkedIn URL with your actual profile.
- Replace project GitHub links with the exact repository URLs.
- Replace `assets/resume.pdf` with your real resume PDF.
- Add any live demo links where available.

## Run Locally

Because this is a static site, you can open `index.html` directly in a browser.

For a local server:

```bash
python3 -m http.server 8080
```

Then visit:

```text
http://localhost:8080
```

## Deploy to GitHub Pages

1. Push this folder to a GitHub repository.
2. Open the repository settings.
3. Go to **Pages**.
4. Set the source to the main branch and root directory.
5. Save and wait for GitHub Pages to publish the site.
